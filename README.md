## 📖 About

This repository is a comprehensive study guide to help you **ace technical interviews**. It covers essential data structures, algorithm patterns, and classic interview problems — each solved in TypeScript, Python, and Go so you can study in your preferred language.

---

## 📂 Topics Covered

### 🗂️ Data Structures
| Topic | Description |
|-------|-------------|
| **Arrays & Strings** | Manipulation, sliding window, two pointers |
| **Linked Lists** | Singly, doubly, cycle detection, reversal |
| **Stacks & Queues** | Implementations, monotonic stack, BFS/DFS helpers |
| **Trees & Graphs** | BST, AVL, BFS, DFS, topological sort |

| **Hash Tables** | Collision handling, frequency maps, caching |

## 🗣️ Languages

Each problem is (or will be) implemented in three languages to maximize learning:

```
📁 <topic>/
 ├── solution.ts     # TypeScript
 ├── solution.py     # Python
 └── solution.go     # Go
```

---

## 🚀 Getting Started

### Prerequisites
- **TypeScript** — Node.js ≥ 18 + `npm install -g ts-node`
- **Python** — Python ≥ 3.9
- **Go** — Go ≥ 1.21

### Run a Solution

```bash
# TypeScript
ts-node <topic>/solution.ts

# Python
python <topic>/solution.py

# Go
go run <topic>/solution.go
```
